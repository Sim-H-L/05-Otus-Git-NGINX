# 05-Otus-Git-NGINX
Учебный репозиторий для Домашней работы по GitHub

Развёрнута ВМ - Ubuntu 24.10
С репозитория установлен git.
Установлен nginx.
Сгенерён ключ.
Директория /etc/nginx/ - инициализирована как локальный репозиторий.
На GitHub создан репозиторий "05-Otus-Git-NGINX".
Инициализирован публичный ключ для авторизации по SSH.
Получена ссылка для связи с локальным репозиторием.
Локально указан репозиторий на GitHUb.
Проверка:
Меняем локально директорию с "index.html" в конфигурационном файле локального nginx.
Делаем первый - "commit".
Заполняем "README.md" на GitHub.
Синхронизируем.
